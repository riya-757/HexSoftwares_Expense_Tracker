# HexSoftwares_Expense_Tracker
# Expense Tracker using Python

This project is a console-based Expense Tracker developed using Python as part of the Python Programming Internship at Hex Software.

## Project Description
The application allows users to add expenses, categorize them, and view total and category-wise expense summaries.

## Features
- Add expenses with date, amount, category, and description
- View all expenses
- Calculate total expenses
- Category-wise expense summary

## Technologies Used
- Python
- Lists and Dictionaries
- Functions
- Conditional Statements
